# DeePM - Private Music Collection

Flutter iOS музыкальный плеер.

## Сборка через FlutLab.io

1. Зайдите на https://flutlab.io
2. Создайте новый проект и загрузите все файлы
3. Положите mp3 файлы в `assets/music/`
4. Нажмите Build > iOS
5. Скачайте IPA файл

## Музыкальные файлы

Положите в `assets/music/`:
- LOVE.mp3
- Roaming.mp3
- Boybend.mp3
- Гикаю.mp3
